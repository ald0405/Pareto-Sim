# Pareto-Sim
Simple Streamlit App

Python modules required 
pandas==1.1.4

streamlit==0.78.0

matplotlib==3.3.3

streamlit==0.78.0

Once installed. 
Open command line in directory where ParetoSimulator.py is stored.
Use commands "streamlit run ParetoSimulator.py"
A window will open in your default web browser
